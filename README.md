# DML Coupon Desktop

DML Coupon Desktop 是一款桌面端优惠券管理工具，支持账号管理、优惠券查看、积分查询、抽奖管理、操作记录、同步记录、公告通知、用户反馈和软件更新。

本仓库仅用于软件下载、版本发布和使用说明，不包含软件源码。

---

| macOS Intel | 支持 | `.dmg` |

## 常见问题

### 为什么优惠券数量和之前看到的不一样？

可能原因包括：

- 优惠券已经使用
- 优惠券已经过期
- 账号状态发生变化
- 同步时间不同
- 平台数据发生变化
- 当前筛选条件不同

建议先确认当前页面筛选条件，再重新同步账号数据。

### 为什么有些账号查询失败？

可能原因包括：

- 网络异常
- 账号信息错误
- 平台限制
- 请求过于频繁
- 账号状态异常

可以稍后重试，或在操作记录、同步记录中查看失败原因。

### Windows 提示未知发布者怎么办？

当前安装包可能没有进行商业代码签名。

如果确认文件来源为本仓库 Release，可以选择继续运行。

### macOS 提示无法验证开发者怎么办？

可以在 macOS 系统设置中手动允许打开。

通常路径为：

```text
系统设置 → 隐私与安全性 → 仍要打开
```

### 软件会自动更新吗？

软件支持检测更新。  
具体是否自动下载或强制更新，以软件内提示为准。

### 可以离线使用吗？

部分页面可以查看本地缓存数据。

但以下功能通常需要网络连接：

- 账号同步
- 积分查询
- 抽奖查询
- 授权验证
- 检测更新
- 提交反馈

### 更换电脑后怎么迁移？

可以使用软件内的数据导出功能，在新电脑上通过导入功能恢复数据。
